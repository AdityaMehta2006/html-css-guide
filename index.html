Deconstructed: A Comprehensive HTML & CSS Guide

This document breaks down the fundamental concepts of HTML and CSS, using the "Aditya Motors" website as a practical example. Each section explains a topic, shows its syntax, and provides code snippets with visual results.

Part 1: HTML Fundamentals

HTML (HyperText Markup Language) is the standard language for creating web pages. It provides the structure and content.

Basic Tags: Headings & Paragraphs

<h1> to <h6> tags define headings, with <h1> being the most important. The <p> tag defines a paragraph.

<h1>Main Title</h1>
<p>This is a paragraph of text.</p>

Attributes

Attributes provide additional information about HTML elements and are always specified in the start tag. They usually come in name/value pairs like name="value".

<a href="https://www.google.com">This is a link</a>
<img src="image.jpg" alt="A descriptive caption">

Formatting

HTML has several elements for defining text with a special meaning.

  • <b> or <strong> - Bold text (strong has semantic importance)
  • <i> or <em> - Italic text (em has semantic importance)
  • <mark> - Marked/highlighted text
  • <u> - Underlined text

Visual Result

Bold Text, Italic Text, Highlighted Text, Underlined Text.

Comments

HTML comments are not displayed in the browser, but they can help document your HTML source code.

<!-- This is a comment. It will not be displayed. -->
<p>This is a normal paragraph.</p>

Images

The <img> tag is used to embed an image in an HTML page. It is an empty tag, meaning it contains attributes only, and does not have a closing tag.

<img src="car.jpg" alt="A luxury sports car" width="500" height="333">

Tables

HTML tables allow you to arrange data into rows and columns. Key tags are <table>, <tr> (table row), <th> (table header), and <td> (table data).

Lists

HTML supports ordered (numbered) and unordered (bulleted) lists.

  • <ul> - An unordered list.
  • <ol> - An ordered list.
  • <li> - A list item.

Links

Links are defined with the <a> (anchor) tag. The destination is specified in the href attribute. You can link text or even wrap an image to make it clickable.

<!-- Text Link -->
<a href="about.html">About Us</a>

<!-- Image Link -->
<a href="details.html">
  <img src="car.jpg" alt="More details">
</a>

Part 2: CSS Fundamentals

CSS (Cascading Style Sheets) is the language we use to style an HTML document. It describes how HTML elements should be displayed.

Benefits & Versions

CSS saves a lot of work. It can control the layout of multiple web pages all at once. The latest version is CSS3, which introduced features like Flexbox, Grid, animations, and transitions.

How to Add CSS

There are three ways to insert CSS:

  1. External CSS: A <link> tag in the <head> section links to an external .css file. This is the best method for styling many pages.
    2. <link rel="stylesheet" href="mystyle.css">
  2. Internal CSS: Defined inside a <style> element in the <head> section. Useful for single-page styles.
    3. <style> body { background-color: linen; } </style>
  3. Inline CSS: Applied directly to an element using the style attribute. Use sparingly.
    4. <h1 style="color:blue; margin-left:30px;">A Blue Heading</h1>

CSS Selectors

Selectors are used to "find" (or select) the HTML elements you want to style.

  • Universal Selector (*): Selects all elements.
  • Type Selector (e.g., h1): Selects elements by their tag name.
  • Class Selector (e.g., .intro): Selects elements with a specific class attribute.
  • ID Selector (e.g., #firstname): Selects one unique element with a specific id attribute.
  • Group Selector (e.g., h1, p): Selects all the elements with the same style definitions.

Part 3: CSS Properties In-Depth

Colors, Backgrounds, and Cursors

You can set colors for text and backgrounds. The cursor property specifies the mouse cursor to be displayed when pointing over an element.

.box {
  color: #333; /* Dark grey text */
  background-color: #f0a500; /* Gold background */
  cursor: pointer; /* Hand cursor on hover */
}

The Box Model

All HTML elements can be considered as boxes. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

Display, Positioning, and Floats

  • display: Specifies how an element is shown. Common values are block, inline, and flex.
  • position: Specifies the type of positioning method used for an element (static, relative, absolute, fixed, or sticky).
  • float: A legacy property used for layout, though modern techniques like Flexbox and Grid are now preferred. It specifies that an element should be placed along the left or right side of its container.

Part 4: Side-by-Side Examples

This section provides direct examples of HTML and the CSS that styles it.

Example 1: Horizontal Navigation Bar

By default, list items (<li>) are block-level elements, meaning they stack vertically. To create a horizontal navigation bar, we change their display property to inline and remove the default list styling.

HTML

<nav>
  <ul class="nav-links">
    <li><a href="#">Home</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

CSS

.nav-links {
  list-style-type: none; /* Removes bullets */
  text-align: center;
}
.nav-links li {
  display: inline; /* Makes items horizontal */
  margin: 0 15px;
}

Example 2: Styling a Data Table

We can greatly improve the readability of a standard HTML table by adding padding, borders, and alternating row colors using the :nth-child(even) pseudo-class.

HTML

<table>
  <tr><th>Model</th><td>Valkyrie</td></tr>
  <tr><th>Engine</th><td>6.5L V12</td></tr>
</table>

CSS

table {
  width: 100%;
  border-collapse: collapse;
}
th, td {
  padding: 12px;
  border-bottom: 1px solid #ddd;
}
tr:nth-child(even) {
  background-color: #fafafa;
}

Example 3: Comprehensive Form Styling

This example shows how to style various form elements, including labels, text inputs, and text areas. It also introduces the :focus pseudo-class, which applies a style (like a colored border) only when the user has clicked into an input field, providing helpful visual feedback.

HTML

<form class="booking-form">
  <label for="name">Full Name:</label>
  <input type="text" id="name">

  <label for="comments">Requests:</label>
  <textarea id="comments" rows="3"></textarea>

  <input type="submit" value="Book Now">
</form>

CSS

.booking-form label {
  font-weight: bold;
  display: block; /* Makes label take its own line */
  margin-top: 15px;
}
.booking-form input[type="text"],
.booking-form textarea {
  width: 100%;
  padding: 10px;
  margin-top: 5px;
  border: 1px solid #ccc;
  border-radius: 6px;
  box-sizing: border-box; /* Important! */
}
/* Style for when user clicks into a field */
.booking-form input:focus,
.booking-form textarea:focus {
  outline: none;
  border-color: #0b4b7d;
  box-shadow: 0 0 5px rgba(11,75,125,0.5);
}
.booking-form input[type="submit"] {
  margin-top: 15px;
  background: #0b4b7d;
  color: white;
  border: none;
  padding: 10px 25px;
  border-radius: 6px;
  cursor: pointer;
}

Visual Result

Example 4: Header with Gradient

The linear-gradient() function creates a smooth color transition. It's part of the background property and is great for creating modern, eye-catching headers.

HTML

<header>
  <h1>Aditya Motors</h1>
  <p>Premium Dealership</p>
</header>

CSS

header {
  background: linear-gradient(to right, #0f2027, #2c5364);
  color: white;
  padding: 30px 0;
  text-align: center;
}
header h1 {
  margin: 0;
  font-size: 36px;
}

Example 5: Clickable Image Card

This common pattern combines an image, text, and a button. The entire card is wrapped in an anchor (<a>) tag to make it a link.

HTML

<a href="#" class="car-card-link">
  <div class="car-card">
    <img src="car.jpg" alt="Luxury Car">
    <div class="content">
      <h4>Pagani Utopia</h4>
      <span class="btn">View Details</span>
    </div>
  </div>
</a>

CSS

.car-card-link {
  text-decoration: none;
  color: inherit;
}
.car-card {
  border-radius: 12px;
  box-shadow: 0 4px 16px rgba(0,0,0,0.1);
  overflow: hidden;
  transition: transform 0.2s ease;
}
.car-card:hover {
  transform: translateY(-5px);
}
.car-card .content { padding: 20px; }
.btn {
  background-color: #f0a500;
  color: white;
  padding: 8px 16px;
  border-radius: 5px;
}
