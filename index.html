Navigation Links

HTML

<nav class="styled-nav">
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

CSS

.styled-nav {
  background-color: #1e293b;
  padding: 0.5rem 1rem;
  border-radius: 0.5rem;
}
.styled-nav ul {
  display: flex;
  gap: 1.5rem;
  list-style: none;
  padding: 0;
}
.styled-nav a {
  color: #a5b4fc;
  text-decoration: none;
  font-weight: 500;
}

Advanced Styling (Gradient & RGBA)

HTML

<div class="gradient-box">
  A box with a gradient background.
</div>

CSS

.gradient-box {
  padding: 2rem;
  border-radius: 0.5rem;
  color: white;
  /* A gradient from a semi-transparent
     indigo to a semi-transparent purple */
  background: linear-gradient(
    90deg, 
    rgba(79, 70, 229, 0.8), 
    rgba(139, 92, 246, 0.8)
  );
}

Embedded Media (Video & iFrame)

HTML

<!-- Video Player -->
<video class="styled-video" controls>
  <source src="movie.mp4" type="video/mp4">
</video>

<!-- iFrame -->
<iframe 
  class="styled-iframe"
  src="https://www.openstreetmap.org/export/embed.html">
</iframe>

CSS

.styled-video, .styled-iframe {
  width: 100%;
  aspect-ratio: 16 / 9;
  border: 1px solid #e2e8f0;
  border-radius: 0.5rem;
  margin-bottom: 1rem;
}

Complex Forms

HTML

<form class="styled-form">
  <label for="email">Email</label>
  <input type="email" id="email">
  
  <label for="msg">Message</label>
  <textarea id="msg" rows="3"></textarea>
  
  <p>Options:</p>
  <div class="radio-group">
    <input type="radio" id="opt1" name="option">
    <label for="opt1">Option 1</label>
  </div>

  <button type="submit">Subscribe</button>
</form>

CSS

.styled-form label {
  display: block;
  margin-bottom: 0.25rem;
  margin-top: 0.75rem;
}
.styled-form input,
.styled-form textarea,
.styled-form button {
  width: 100%;
  background: #f1f5f9;
  border: 1px solid #e2e8f0;
  padding: 0.5rem;
  border-radius: 0.25rem;
}
.radio-group {
  display: flex;
  align-items: center;
  gap: 0.5rem;
}
.radio-group input {
  width: auto; /* Override full width */
}
.styled-form button {
  margin-top: 1rem;
  background: #4f46e5;
  color: white;
}

Flexbox Layout & Cursors

HTML

<div class="flex-container">
  <div class="flex-item pointer">Pointer</div>
  <div class="flex-item help">Help</div>
  <div class="flex-item wait">Wait</div>
</div>

CSS

.flex-container {
  display: flex;
  justify-content: space-around;
  align-items: center;
  background-color: #f1f5f9;
  padding: 1rem;
  border-radius: 0.5rem;
}
.flex-item {
  background-color: #4f46e5;
  color: white;
  padding: 1rem;
  border-radius: 0.5rem;
}
.pointer { cursor: pointer; }
.help { cursor: help; }
.wait { cursor: wait; }
