Internet Fundamentals

History of the Internet

The internet began in the 1960s as ARPANET, a US military project to create a decentralized communication network. Key milestones include the invention of TCP/IP in the 1970s, the Domain Name System (DNS) in 1983, and the creation of the World Wide Web by Tim Berners-Lee in 1990, which made the internet accessible to the public.

Client-Server Model

This model is the foundation of the web. A **Client** (like your web browser) requests resources from a **Server**, which is a powerful computer that stores files and data. The server processes the request and sends a response back to the client.

Web Servers & Web Services

  • Web Server: Software (like Apache or Nginx) that runs on a server computer. Its primary job is to accept HTTP requests from clients (browsers) and send back the requested files (HTML, CSS, images).
  • Web Services: A way for different applications to communicate with each other over the internet, often through an API. They allow a website to use functionality from another service (e.g., using Google Maps on your site).

HTML Concepts

HTML vs HTML5

HTML5 is the latest version of HTML. It introduced new features designed for modern web applications, including new semantic tags like <article> and <section>, native support for <audio> and <video>, and new form input types.

CSS Positioning

The position property specifies the type of positioning method used for an element.

  • static: Default value. Elements render in order.
  • relative: Positioned relative to its normal position.
  • absolute: Positioned relative to its nearest positioned ancestor.
  • fixed: Positioned relative to the viewport (stays in place on scroll).

Basic & Structural Tags

<h1> to <h6>: Define hierarchical headings.

<p>: Defines a paragraph.

<div>: A generic block-level container for styling.

<span>: A generic inline container for styling text.

<header>: Defines the header for a document or section.

<footer>: Defines the footer for a document or section.

<nav>: Defines a block of navigation links.

Links & Media Tags

<a href="...">: Defines a hyperlink.

<img src="..." alt="...">: Embeds an image.

<iframe src="...">: Embeds another HTML document.

Table Tags

<table>: The wrapper element for a table.

<thead>: Groups the header content in a table.

<tbody>: Groups the body content in a table.

<tr>: Defines a row in a table.

<th>: Defines a header cell in a table.

<td>: Defines a standard data cell in a table.

What is the difference between a Web Server and a Web Service?

A **Web Server** is software that serves website files to a browser (e.g., Apache). A **Web Service** is a function that a website can call over the internet to get data or perform an action (e.g., a weather API).

What is the purpose of `` and `` in a table?

They group the header and body content of a table, providing structure for styling and accessibility. They also allow browsers to scroll the table body independently of the header.

What is the difference between `em` and `rem` units?

em is relative to the font-size of its direct parent element. rem (root em) is relative to the font-size of the root `` element. Using `rem` for font sizes is often preferred for more predictable scaling.

What is CSS Specificity?

Specificity is the algorithm browsers use to determine which CSS rule is applied if multiple rules target the same element. Rules with more specific selectors (like an ID #my-id) will override rules with less specific selectors (like a class .my-class).
