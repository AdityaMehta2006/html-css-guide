Deconstructed: A Guide to the Aditya Motors Web Page

This document breaks down the HTML and CSS used to create the "Aditya Motors" website. Each section explains a specific tag or property, shows its syntax, provides the exact code from your project, and includes a visual example of the result.

Part 1: Core HTML Structure

These tags form the main layout and structural containers of the page.

<header>

Definition

Represents introductory content, typically a group of introductory or navigational aids. It often contains <h1> tags, logos, and search forms.

Syntax

<header> ... content ... </header>

Code Snippet

<header>
  <h1>Aditya Motors</h1>
  <p>Your Destination for Hypercars and Luxury Test Drives</p>
</header>

Visual Result

Aditya Motors

Your Destination for Hypercars and Luxury Test Drives

<nav>

Definition

Represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents.

Syntax

<nav> ... navigation links ... </nav>

Code Snippet

<nav>
  <ul class="nav-links">
    <li><a href="index.html">Home</a></li>
    ...
  </ul>
</nav>

Visual Result

Home Contact Location

<div>

Definition

The generic container for flow content. It has no effect on the content or layout until styled using CSS. It's used to group elements for styling purposes or because they share attribute values.

Syntax

<div class="your-class-name"> ... content ... </div>

Code Snippet

<div class="section">
  <h2>Aston Martin Valkyrie</h2>
  ...
</div>

Visual Result

<h1>, <h2>

Definition

Heading tags, used to define a hierarchy of titles and subtitles. <h1> is the most important, <h2> is a subsection, and so on, down to <h6>.

Syntax

<h1>Main Title</h1>

Visual Result

<iframe>

Definition

Used to embed another HTML document within the current one. Commonly used for embedding maps, social media feeds, or videos.

Syntax

<iframe src="url" width="x" height="y" title="description"></iframe>

Code Snippet

<iframe width="700" height="394" src="https://www.youtube.com/embed/a57WNM2Yhc8" title="Aston Martin Valkyrie"></iframe>

Visual Result

<table>, <tr>, <th>, <td>

Definition

  • <table>: The container for tabular data.
  • <tr>: A row in the table.
  • <th>: A header cell in the table (bold and centered by default).
  • <td>: A standard data cell in the table.

Syntax

<table> <tr> <th>Header</th> <td>Data</td> </tr> </table>

Code Snippet

<table>
  <tr>
    <th>Model</th>
    <td>Aston Martin Valkyrie</td>
  </tr>
  <tr>
    <th>Engine</th>
    <td>6.5L V12 Hybrid</td>
  </tr>
</table>

Part 3: HTML Forms

These tags are used to collect input from the user.

<form>, <label>, <input>, <textarea>

Definition

  • <form>: A container for form controls.
  • <label>: A caption for an item in a user interface.
  • <input>: A typed data field, used to create interactive controls. Its type attribute is critical (e.g., text, tel, radio, submit).
  • <textarea>: A multi-line plain-text editing control.

Syntax

<form> <label for="id">Label</label> <input type="text" id="id"> </form>

Code Snippet

<form>
  <label for="name">Full Name:</label>
  <input type="text" id="name" name="name" required>

  <input type="radio" id="valkyrie" name="model" value="Valkyrie">
  <label for="valkyrie">Valkyrie</label>
</form>

Visual Result

Part 4: Core CSS Properties

These CSS properties define the fundamental look and feel of the page.

font-family, color, background-color

Definition

  • font-family: Specifies a prioritized list of one or more font family names.
  • color: Sets the foreground color value of an element's text.
  • background-color: Sets the background color of an element.

Syntax

selector {
  font-family: "Font Name", sans-serif;
  color: #333;
  background-color: #f0f2f5;
}

Code Snippet

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f0f2f5;
  color: #333;
}

Visual Result

This text uses the 'Segoe UI' font on a light grey background with dark grey text.

margin, padding

Definition

The core of the Box Model.

  • margin: Sets the space on the outside of an element's border.
  • padding: Sets the space on the inside of an element's border, between the border and the content.

Syntax

selector { margin: 10px; padding: 20px; }

Code Snippet

.section {
  padding: 25px;
  margin: 40px auto;
}

Visual Result

The dashed border is the edge of the element. The space inside is padding, the space outside is margin.

:hover, transition

Definition

  • :hover: A pseudo-class that applies styles when the user's mouse pointer is over an element.
  • transition: Defines how a CSS property change should animate smoothly from one state to another.

Syntax

selector { transition: property duration timing-function; }
selector:hover { property: new-value; }

Code Snippet

.nav-links a {
  transition: color 0.3s ease;
}
.nav-links a:hover {
  color: #f0a500;
}

Visual Result

Hover over this link

Part 5: Advanced CSS Selectors & Properties

These selectors and properties add more specific and advanced styling to the page.

Advanced Selectors

Definition

Advanced selectors allow for more precise targeting of elements.

  • Attribute Selector [attr="value"]: Selects elements based on an attribute and its value.
  • Pseudo-class Selector :nth-child(): Selects elements based on their position among siblings.

Syntax

selector[attribute="value"] { ... }
selector:pseudo-class { ... }

Code Snippet

input[type="text"] {
  width: 100%;
}

tr:nth-child(even) {
  background-color: #fafafa;
}

Visual Result

The text input below is styled because its type attribute is "text". In the table, every even row (2nd, 4th, etc.) has a light grey background.

linear-gradient()

Definition

Creates an image consisting of a progressive transition between two or more colors along a straight line. It is used with the background property.

Syntax

selector { background: linear-gradient(direction, color1, color2, ...); }

Code Snippet

header {
  background: linear-gradient(to right, #0f2027, #203a43, #2c5364);
}

Visual Result

This element has a horizontal gradient background.

border-radius & box-shadow

Definition

  • border-radius: Rounds the corners of an element's outer border edge.
  • box-shadow: Adds shadow effects around an element's frame.

Syntax

selector {
  border-radius: 12px;
  box-shadow: offsetX offsetY blurRadius color;
}

Code Snippet

.section {
  border-radius: 12px;
  box-shadow: 0 4px 16px rgba(0, 0, 0, 0.1);
}

Visual Result

This box has rounded corners and a subtle shadow.
